    Definition 1.1.7. ordinal.

    a set is transitive if x \(\in\) T implies x \(\subset\) T. A set x is an ordinal if it is transitive and well-ordered by \(\in\text{.}\) define \(x+1 = x \cup \{x\}\text{.}\) this is another ordinal called the successor of x. x is called a successor ordinal if there exist a y such that x=y+1. the small ordinal is \(\emptyset\) or 0. if x is not 0 and not a successor ordinal, x is a limit ordinal.
    \begin{equation*} x = \bigcup_{\gamma \in x}\gamma \end{equation*}
    the first limit ordinal is \(\omega\text{.}\) first uncountable ordinal is \(\omega_1\text{,}\) the set of all countable ordinals. the collection of all ordinals is a proper class. it is well ordered by \(\in\text{,}\) meaning, any nonempty set of ordinals has a least element. given a set of ordinals A, its supremum is defined to be, \(sup_{a \in A} a= \bigcup_{a \in A} a\text{.}\) it is the least ordinal bigger or equal to all \(a \in A\) given any well ordered set (S, \(\lt\)), there is a unique ordinal a such that (S, \(\lt\)) is isomorphic to (a, \(\in\)), this is called the order type of the well ordered set.

    Definition 1.1.9. module.

    A left R-module M consists of an abelian group (M, +) and an operation \(\cdot : R \times M \rightarrow M\) such that for all r, s in R and x, y in M, we have
    \begin{equation*} r \cdot (x+y) = r \cdot x + r \cdot y \end{equation*}
    \begin{equation*} (r + s) \cdot x = r \cdot x + s \cdot x \end{equation*}
    \begin{equation*} rx \cdot x = r \cdot (s \cdot x) \end{equation*}
    \begin{equation*} 1 \cdot x = x \end{equation*}
