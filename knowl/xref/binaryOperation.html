Definition 1.1.34. binary operation.

Takes two elements and gives one. If the sets taken and given out is not specified, but the binary operation is specified over only one set S, assume the binary operation \(\cdot\) is an infix operator which takes two elements of that set to that set. \((\cdot): S \times S \rightarrow S\text{.}\)
