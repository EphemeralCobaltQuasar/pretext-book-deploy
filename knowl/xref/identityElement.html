Definition 1.1.37. identity element.

A set S with a binary operation \(\cdot\) has an identity element if there exist an element e such that, for all a in S, \(a \cdot e=e \cdot a\text{.}\) The identity element is sometimes 0 (when the binary operation is +), or 1 (when the binary operation is \(\cdot\))
